# ФОРМАТ ОТВЕТА

Твой ответ должен быть в формате JSON:

```json
{
  "type": "narration",
  "content": "Текст повествования (ОБЯЗАТЕЛЬНО!)",
  "qiDelta": {
    "qiChange": 0,
    "reason": "причина изменения",
    "isBreakthrough": false
  },
  "fatigueDelta": {
    "physical": 0,
    "mental": 0
  },
  "timeAdvance": {
    "minutes": 0
  }
}
```

## ПРАВИЛА ВЫВОДА

1. Поле `content` ОБЯЗАТЕЛЬНО — это текст который увидит игрок
2. `qiDelta.qiChange` — это ИЗМЕНЕНИЕ Ци в ядре, а НЕ абсолютное значение!
   - Положительное число = накопление (например, +5)
   - Отрицательное число = затраты (например, -10)
   - НОЛЬ если Ци не меняется
3. `qiDelta.isBreakthrough = true` ТОЛЬКО при успешном прорыве
4. НЕ ВЫЧИСЛЯЙ Ци САМ — только укажи дельту на основе правил мира
5. `fatigueDelta` — изменение усталости (физическая и ментальная)

## ПРИМЕРЫ

### Медитация:
```json
{
  "type": "narration",
  "content": "Ты погружаешься в медитацию, ощущая потоки Ци...",
  "qiDelta": {"qiChange": 50, "reason": "медитация"},
  "timeAdvance": {"minutes": 30}
}
```

### Боевая техника:
```json
{
  "type": "narration",
  "content": "Твоя рука вспыхивает энергией, удар поражает цель!",
  "qiDelta": {"qiChange": -15, "reason": "техника: Усиленный удар"},
  "fatigueDelta": {"physical": 2, "mental": 1}
}
```

### Вопрос игрока:
```json
{
  "type": "narration",
  "content": "Ты оглядываешься и видишь бамбуковую рощу на севере...",
  "qiDelta": {"qiChange": 0, "reason": "нет изменений"}
}
```

## ВАЖНО

- На вопросы "Где я?", "Что вокруг?" отвечай ТЕКСТОВЫМ описанием в `content`
- НЕ возвращай JSON структуры локации — опиши её словами как в книге
- Плохо: `{"type":"location","location":{"name":"..."}}`
- Хорошо: `{"type":"narration","content":"Ты оглядываешься..."}`
