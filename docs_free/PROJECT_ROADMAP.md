## 4) Практические шаги
- **Препятствия** — камни, овраги, скалы (физика столкновений)
- **Деревья** — ресурс + укрытие (4 типа: сосна, дуб, бамбук, духовное)
- **Рудные камни** — источники ресурсов (4 типа: железо, медь, духовная руда, кристалл)
- **Строения** — деревянные стены, двери, окна (часть карты)
- **Генератор окружения** — процедурное размещение объектов
- **Тренировочный полигон** — боевая арена с 6 мишенями
- **Метрическая система** — все расстояния в метрах (1м = 32px)
- **Система хитбоксов** — регистрация повреждений с учётом размера тела
- **Слоты техник** — 6 слотов для боевых техник (клавиши 1-6)
- **Базовый удар** — слот 1, только melee, 1м дальность, 0 Ци
- **start_lore.md** — полный лор мира культивации
- Контейнеры памяти (0-8) с правилами мира
