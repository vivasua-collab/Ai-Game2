## 🎯 ЦЕЛЬ

Реализовать **независимый тик-таймер**, который:
- Работает в фоне, не завися от действий игрока
- Тикает каждые 1 секунду реального времени (ФИКСИРОВАНО)
- Переменная: сколько ИГРОВОГО времени проходит за тик
- Синхронизирован между React UI и Phaser Game

---

```
┌─────────────────────────────────────────────────────────────────┐
│   1 ТИК = 1 СЕКУНДА РЕАЛЬНОГО ВРЕМЕНИ (НЕИЗМЕННО)              │
│                                                                 │
│   Переменная: ИГРОВОЕ ВРЕМЯ за 1 тик (minutesPerTick)           │
└─────────────────────────────────────────────────────────────────┘
```

| Скорость | Игровое время/тик | За 1 мин реального | Применение |
|----------|-------------------|--------------------|------------|

## ⚠️ КРИТИЧЕСКИЕ ПРОБЛЕМЫ (Phase 8)

### 1. Phaser Calendar Desync
- **Проблема:** `LocationScene.currentGameTime` не имеет year/month/season
- **Решение:** Обновить до полного `GameTime` интерфейса

### 2. Movement During Pause
- **Проблема:** `handleMovement()` не проверяет `isPaused`
- **Решение:** Добавить проверку в начале функции

### 3. No Movement Time Cost
- **Проблема:** WASD движение не связано со временем
- **Решение:** Создать MovementProcessor с привязкой к тикам
