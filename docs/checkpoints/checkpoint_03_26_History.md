### Архитектура "Божество → Облако → Земля"

```
👁️ БОЖЕСТВО (Игрок)
   └── Управляет аватаром, принимает решения

☁️ ОБЛАКО (Браузер / Thin Client)
   └── Отображает мир, передаёт волю божества
   └── Phaser только визуализация!

🌍 ЗЕМЛЯ (Сервер - TruthSystem)
   └── Хранит состояние, исполняет расчёты
   └── NPC AI, боёвка, техники, инвентарь
   └── 1 TICK = 1 СЕКУНДА реального времени
```
