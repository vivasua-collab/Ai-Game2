## 4) Практические шаги
- **Dialog** — компонент из shadcn/ui
- **Модальное окно** — общее название
- **DialogContent** — контейнер содержимого
- **DialogHeader** — заголовок окна
- **DialogTitle** — текст заголовка
- Часть интерфейса, занимающая область экрана
- Может быть внутри диалога или самостоятельной
- **TabsList** — контейнер кнопок вкладок
- **TabsTrigger** — кнопка вкладки
- **TabsContent** — содержимое вкладки
- Активный элемент для действия
- Варианты: primary, secondary, outline, ghost
