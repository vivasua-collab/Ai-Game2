## Анализ

### Текущие проблемы

#### NPC: elf (Уровень 7)
- Сила: 11 (слишком мало для ур.7)
- Ловкость: 9 (слишком мало для ур.7)
- Ци: 1266 (слишком мало для ур.7)
- Экипировка: отсутствует

### Проблема 1: baseStats не учитывают уровень
**Текущая логика:**
```
stats = random(species.baseStats) + role.modifiers
```

**Elf baseStats:**
- strength: { min: 3, max: 15 }
- agility: { min: 8, max: 30 }
- intelligence: { min: 10, max: 35 }
- vitality: { min: 4, max: 15 }

**Проблема:** Эти значения не масштабируются с уровнем!

### Проблема 2: coreCapacity слишком маленький
**Текущая логика:**
```
coreCapacity = baseCapacity * (1 + level * 0.1)
```

Для уровня 7: `baseCapacity * 1.7` - только +70%!

### Проблема 3: Экипировка не генерируется
Нужно проверить `generateEquipment()` в npc-generator.ts

## Работа

### [1] Проверка Dialog component
**Статус:** ✅ Исправлено

**Проблема:** DialogContent имел `max-w-[calc(100%-2rem)] sm:max-w-lg` по умолчанию

**Решение:** Добавлены `!max-w-6xl !w-[95vw] overflow-hidden` для принудительного переопределения:
```tsx
<DialogContent className="bg-slate-900 border-slate-700 text-white !max-w-6xl !w-[95vw] max-h-[90vh] p-0 overflow-hidden">
```

### [2] Отображение ID NPC
**Статус:** ✅ Добавлено

**Изменение:** Добавлена строка с ID под именем NPC:
```tsx
<div className="text-xs text-slate-500 font-mono mt-1">ID: {selectedNPC.id}</div>
```

### [3] Баланс характеристик NPC по уровням
**Статус:** ✅ Исправлено

**Проблема:** generateStats() не учитывал уровень культивации

**Решение:** Добавлены множители характеристик по уровням:
```typescript
const levelMultipliers: Record<number, number> = {
  1: 1.0, 2: 1.3, 3: 1.6, 4: 2.0, 5: 2.5,
  6: 3.0, 7: 3.5, 8: 4.0, 9: 5.0,
};
```

**Изменён порядок генерации:**
1. species
2. role
3. **cultivation** (ПЕРЕД статами!)
4. stats (с учётом cultivation.level)

### [4] Развитие ядра Ци
**Статус:** ✅ Исправлено

**Проблема:** coreCapacity = baseCapacity * (1 + level * 0.1) - только +70% для уровня 7

**Решение:** Добавлены множители coreCapacity по уровням:
```typescript
const coreCapacityMultipliers: Record<number, number> = {
  1: 1.0, 2: 2.0, 3: 3.5, 4: 5.0, 5: 7.5,
  6: 10.0, 7: 15.0, 8: 20.0, 9: 30.0,
};
```

**Результат для уровня 7:**
- Было: `baseCapacity * 1.7` (~1700)
- Стало: `baseCapacity * 15.0` (~15000-37500)

### [5] Экипировка NPC
**Статус:** ✅ Добавлено

**Проблема:** equipment всегда был пустым `{}`

**Решение:** Добавлена функция generateEquipment():
- Читает role.equipment
- Создаёт слоты: weapon, armor, helmet, boots, gloves, accessory1, accessory2
- Категории (category_*) оставляют null для заполнения из пула
- Конкретные ID используются напрямую
