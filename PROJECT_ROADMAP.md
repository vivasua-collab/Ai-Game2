## 🔮 Планы развития

### 🎮 Режимы интерфейса (Admin / Player)

**Приоритет: Средний**

**Идея:** Разделить интерфейс на два режима:

1. **Режим Администратора**
   - Доступ ко всем настройкам системы
   - Управление LLM провайдерами
   - Полный доступ к базе данных (сброс, миграции)
   - Просмотр системных логов
   - Управление бэкапами
   - Настройка параметров игры

2. **Режим Игрока**
   - Только игровой интерфейс
   - Минимальные настройки (громкость, тема)
   - Нет доступа к админ-функциям
   - Упрощённый стартовый экран

**Реализация:**
```
/src/app/admin/     - Админ-панель (защищённый маршрут)
/src/app/play/      - Игровой интерфейс
/src/middleware.ts  - Проверка прав доступа
```

**Статус:** 🔵 Идея (не начато)
