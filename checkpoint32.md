## 📊 Ключевые решения

### 1. Типы NPC

| Тип | ID формат | Хранение | Использование |
|-----|-----------|----------|---------------|
| **Preset NPC** | `NPC_PRESET_XXXXX` | БД (Prisma) + JSON пресеты | Сюжетные персонажи, квестодатели |
| **Generated NPC** | `TEMP_XXXXXX` | Память сессии | Статисты, монстры, фоновые NPC |

### 2. Иерархия принадлежности

```
Государство (Nation)
    └── Фракция (Faction) — Альянс сект
        └── Секта (Sect)
            └── Культиватор (Character/NPC)
                └── Одеяние секты (Sect Attire)
```

### 3. Уровни отношений

| Уровень | Вес | Пример |
|---------|-----|--------|
| Личный | 100% | NPC лично знает игрока |
| Сектовый | 50% | Секта NPC враждует с сектой игрока |
| Фракционный | 30% | Фракции идеологически против |
| Государственный | 20% | Государства в холодной войне |

### 4. Одеяние секты (Sect Attire)

- Слот: `torso` (основное одеяние)
- Дополнительные элементы: `badge`, `sash` (accessory слоты)
- Эффекты:
  - `+disposition` с членами той же секты
  - `-disposition` с врагами секты
  - Визуальная идентификация (цвета, символы)

## 📊 Новые Prisma модели

### Nation (Государство)
```prisma
model Nation {
  id, name, description
  governmentType: monarchy | republic | theocracy | federation | warlord
  capitalLocationId
  relations: JSON      // { nationId: disposition }
  officialSects: JSON  // [sectId, ...]
  undergroundSects: JSON
}
```

### Faction (Фракция)
```prisma
model Faction {
  id, name, description
  ideology: righteous | demonic | neutral | pragmatic | isolationist
  leaderSectId
  memberSectIds: JSON
  relations: JSON      // { factionId: disposition }
}
```

### Sect (расширение)
```prisma
model Sect {
  // ... существующие поля
  nationId, factionId
  sectType: orthodox | unorthodox | demonic | neutral | scholarly | martial
  standing: official | underground | exiled | nomadic | independent
  relations: JSON      // SectRelations
  insignia: JSON       // SectInsignia
  atWar: JSON, allied: JSON
}
```

### NPC (расширение)
```prisma
model NPC {
  // ... существующие поля
  isPreset: Boolean
  presetId: String?
  relations: JSON      // NPCRelations
}
```

### InventoryItem (расширение)
```prisma
model InventoryItem {
  // ... существующие поля
  isSectAttire: Boolean
  sectId: String?
  attireVisuals: JSON
  socialEffects: JSON
}
```
