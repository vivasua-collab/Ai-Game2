## 1️⃣ Архитектура Слоёв Защиты

### 1.1 Полный порядок прохождения урона

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                    ПОРЯДОК ПРОХОЖДЕНИЯ УРОНА (10 СЛОЁВ)                             │
├─────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                      │
│  ╔═════════════════════════════════════════════════════════════════════════════╗    │
│  ║  СЛОЙ 1: ИСХОДНЫЙ УРОН                                                       ║    │
│  ║  rawDamage = techniqueDamage × qiDensity × gradeMult × elementMult            ║    │
│  ╚═════════════════════════════════════════════════════════════════════════════╝    │
│                                      ↓                                               │
│  ╔═════════════════════════════════════════════════════════════════════════════╗    │
│  ║  СЛОЙ 2: ОПРЕДЕЛЕНИЕ ЧАСТИ ТЕЛА                                              ║    │
│  ║  hitPart = rollBodyPart(attacker, target, weapon)                            ║    │
│  ║  head: 5%, torso: 40%, heart: 2%, arm: 20%, leg: 24%, hands: 8%, feet: 1%   ║    │
│  ╚═════════════════════════════════════════════════════════════════════════════╝    │
│                                      ↓                                               │
│  ╔═════════════════════════════════════════════════════════════════════════════╗    │
│  ║  СЛОЙ 3: АКТИВНАЯ ЗАЩИТА (выбор игрока)                                       ║    │
│  ║  ├── Уклонение: dodgeChance = 5% + (AGI-10) × 0.5% - armorDodgePenalty       ║    │
│  ║  │   └── Успех → damage = 0, END                                              ║    │
│  ║  ├── Парирование: parryChance = weaponParryBonus + (AGI-10) × 0.3%           ║    │
│  ║  │   └── Успех → damage ×= (1 - blockEffectiveness), durability--            ║    │
│  ║  └── Блок щитом: blockChance = shieldBlock + (STR-10) × 0.2%                 ║    │
│  ║      └── Успех → damage ×= (1 - shieldEffectiveness), shieldDurability--     ║    │
│  ╚═════════════════════════════════════════════════════════════════════════════╝    │
│                                      ↓                                               │
│  ╔═════════════════════════════════════════════════════════════════════════════╗    │
│  ║  СЛОЙ 4: БУФЕР ЦИ (только для техник Ци) — НОВАЯ МЕХАНИКА                     ║    │
│  ║  if (isQiTechnique && currentQi > minQiForBuffer) {                          ║    │
│  ║    absorptionRatio = hasShieldTechnique ? 1.0 : 3.0                          ║    │
│  ║    requiredQi = damage × absorptionRatio                                     ║    │
│  ║    if (currentQi >= requiredQi) {                                            ║    │
│  ║      qiConsumed = requiredQi                                                 ║    │
│  ║      damage = 0, END ← ПОЛНОЕ ПОГЛОЩЕНИЕ                                     ║    │
│  ║    } else {                                                                  ║    │
│  ║      qiConsumed = currentQi                                                  ║    │
│  ║      damage = damage - (currentQi / absorptionRatio)                         ║    │
│  ║    }                                                                         ║    │
│  ║  }                                                                           ║    │
│  ╚═════════════════════════════════════════════════════════════════════════════╝    │
│                                      ↓                                               │
│  ╔═════════════════════════════════════════════════════════════════════════════╗    │
│  ║  СЛОЙ 5: БРОНЯ ЧАСТИ ТЕЛА (покрытие)                                         ║    │
│  ║  armor = getEquippedArmorForPart(hitPart)                                    ║    │
│  ║  if (random() < armor.coverage) {                                            ║    │
│  ║    // Броня покрывает удар                                                   ║    │
│  ║    goto СЛОЙ 6                                                               ║    │
│  ║  } else {                                                                    ║    │
│  ║    // Броня НЕ покрывает удар — урон идёт напрямую в СЛОЙ 7                  ║    │
│  ║  }                                                                           ║    │
│  ╚═════════════════════════════════════════════════════════════════════════════╝    │
│                                      ↓                                               │
│  ╔═════════════════════════════════════════════════════════════════════════════╗    │
│  ║  СЛОЙ 6: СНИЖЕНИЕ УРОНА БРОНЁЙ                                               ║    │
│  ║  // Процентное снижение                                                      ║    │
│  ║  damageReduction = armor.damageReduction + materialBonus + gradeBonus         ║    │
│  ║  damage ×= (1 - damageReduction)                                             ║    │
│  ║                                                                              ║    │
│  ║  // Плоское вычитание (пробитие)                                             ║    │
│  ║  penetration = weapon.penetration + attackerSTR × 0.5 + techniquePenetration  ║    │
│  ║  effectiveArmor = max(0, armor.armor - penetration)                          ║    │
│  ║  damage = max(1, damage - effectiveArmor × 0.5)                              ║    │
│  ║                                                                              ║    │
│  ║  // Износ брони                                                              ║    │
│  ║  armor.durability.current -= damage × 0.1 / hardness                         ║    │
│  ╚═════════════════════════════════════════════════════════════════════════════╝    │
│                                      ↓                                               │
│  ╔═════════════════════════════════════════════════════════════════════════════╗    │
│  ║  СЛОЙ 7: МАТЕРИАЛ ТЕЛА ЦЕЛИ                                                  ║    │
│  ║  materialReduction = getMaterialReduction(target.bodyMaterial)                ║    │
│  ║  damage ×= (1 - materialReduction)                                           ║    │
│  ║                                                                              ║    │
│  ║  │ Материал    │ Твёрдость │ Снижение урона │                               ║    │
│  ║  │ flesh       │ 3         │ 0%             │                               ║    │
│  ║  │ scaled      │ 6         │ 30%            │                               ║    │
│  ║  │ mineral     │ 8         │ 50%            │                               ║    │
│  ║  │ ethereal    │ 1         │ 70% (физ)      │                               ║    │
│  ║  │ chaos       │ 5         │ 20%            │                               ║    │
│  ╚═════════════════════════════════════════════════════════════════════════════╝    │
│                                      ↓                                               │
│  ╔═════════════════════════════════════════════════════════════════════════════╗    │
│  ║  СЛОЙ 8: ИТОГОВЫЙ УРОН                                                        ║    │
│  ║  finalDamage = max(1, floor(damage))                                         ║    │
│  ╚═════════════════════════════════════════════════════════════════════════════╝    │
│                                      ↓                                               │
│  ╔═════════════════════════════════════════════════════════════════════════════╗    │
│  ║  СЛОЙ 9: РАСПРЕДЕЛЕНИЕ ПО HP (Kenshi-style)                                   ║    │
│  ║  redHP -= damage × 0.7     // 70% в функциональную HP                         ║    │
│  ║  blackHP -= damage × 0.3   // 30% в структурную HP                            ║    │
│  ║                                                                              ║    │
│  ║  if (redHP <= 0) → паралич части тела                                         ║    │
│  ║  if (blackHP <= 0) → отрубание части тела                                     ║    │
│  ╚════════════════════════════════════════════════════════════ │
│                                      ↓                                               │
│  ╔═════════════════════════════════════════════════════════════════════════════╗    │
│  ║  СЛОЙ 10: ПОСЛЕДСТВИЯ                                                         ║    │
│  ║  ├── Кровотечение: если damage > threshold → bleedDamage per tick            ║    │
│  ║  ├── Шок: если redHP упала ниже 30% → штрафы к действиям                     ║    │
│  ║  ├── Оглушение: шанс = damage / maxHP × 10%                                  ║    │
│  ║  └── Смерть: если heart HP ≤ 0 или head HP ≤ 0                               ║    │
│  ╚═════════════════════════════════════════════════════════════════════════════╝    │
│                                                                                      │
└─────────────────────────────────────────────────────────────────────────────────────┘
```

### 3.4 Формула итоговой защиты

```typescript
function calculateEffectiveArmor(armor: Armor): number {
  const baseArmor = armor.baseStats.armor;
  const materialMult = armor.material.properties.defenseBonus / 100 + 1;
  const gradeMult = GRADE_CONFIGS[armor.grade].effectivenessMultiplier;
  
  // Состояние прочности
  const conditionMult = getConditionMultiplier(armor.durability);
  
  return Math.floor(baseArmor * materialMult * gradeMult * conditionMult);
}

function calculateDamageReduction(armor: Armor): number {
  const baseDR = armor.baseStats.damageReduction || 0;
  const materialDR = armor.material.properties.resistanceBonus?.physical || 0;
  const gradeDR = GRADE_CONFIGS[armor.grade].damageReductionBonus || 0;
  
  // Мягкий кап: максимум 80% снижения урона
  return Math.min(0.80, (baseDR + materialDR + gradeDR) / 100);
}
```

## 4️⃣ Штрафы от Брони

### 4.1 Весовые штрафы

```typescript
interface WeightPenalties {
  speedPenalty: number;      // -% скорости
  dodgePenalty: number;      // -% уклонения
  staminaCostPenalty: number; // +% стоимости выносливости
}

function calculateWeightPenalties(totalWeight: number, characterSTR: number): WeightPenalties {
  const baseCapacity = 10 + (characterSTR - 10) * 2;  // кг
  const overload = Math.max(0, totalWeight - baseCapacity);
  
  return {
    speedPenalty: Math.min(50, overload * 0.5),         // до -50%
    dodgePenalty: Math.min(40, overload * 0.3),         // до -40%
    staminaCostPenalty: Math.min(100, overload * 1.0),  // до +100%
  };
}
```

### 4.2 Штрафы к проводимости Ци

| Тип материала | Штраф к qi_efficiency |
|---------------|----------------------|
| Ткань (cloth) | 0% |
| Кожа (leather) | -5% |
| Железо (iron) | -10% |
| Сталь (steel) | -15% |
| Холодное железо | -20% |
| Духовное железо | +5% (бонус!) |
| Нефрит | +10% (бонус!) |

### 4.3 Пример расчёта штрафов

```
Практик L8, Сила 45
Полный комплект брони: 35 кг

Базовая вместимость: 10 + (45-10) × 2 = 80 кг
Перегруз: max(0, 35 - 80) = 0 кг

Результат: НЕТ ШТРАФОВ (броня лёгкая для практика L8)

Если бы Сила была 15:
Базовая вместимость: 10 + (15-10) × 2 = 20 кг
Перегруз: 35 - 20 = 15 кг

Штрафы:
• Скорость: -15 × 0.5 = -7.5%
• Уклонение: -15 × 0.3 = -4.5%
• Выносливость: +15 × 1.0 = +15%
```

## 9️⃣ Износ Брони при Блокировании

### 9.1 Формула потери прочности

```typescript
/**
 * Расчёт потери прочности брони при блокировании урона
 * 
 * Принцип: Броня "принимает на себя" часть урона и изнашивается
 */
function calculateDurabilityLoss(
  damageAbsorbedByArmor: number,  // Урон, который броня заблокировала
  armorHardness: number,          // Твёрдость материала (1-10)
  armorCondition: number          // Процент прочности (0-100)
): number {
  // Базовая потеря: 0.1% макс. прочности за каждую 1000 урона
  const baseLoss = damageAbsorbedByArmor / 10000;  // 10,000 урона = 1% потери
  
  // Твёрдость снижает износ (твёрдость 10 = 50% износа)
  const hardnessFactor = 1 - (armorHardness - 1) / 18;  // 1.0 → 0.94, 10.0 → 0.5
  
  // Повреждённая броня изнашивается быстрее
  const conditionFactor = 1 + (1 - armorCondition / 100) * 0.5;  // до ×1.5 при 0%
  
  // Итоговая потеря
  const totalLoss = baseLoss * hardnessFactor * conditionFactor;
  
  return totalLoss;  // В процентах от maxDurability
}
```

### 9.2 Пример износа

```
Броня: Нагрудник "Кираса Драконьей Чешуи"
• Твёрдость: 9 (кость дракона)
• Прочность: 510/510 (100%)
• Заблокированный урон: 18,000

Расчёт:
  baseLoss = 18,000 / 10,000 = 1.8%
  hardnessFactor = 1 - (9-1)/18 = 0.556
  conditionFactor = 1.0 (броня целая)
  
  totalLoss = 1.8% × 0.556 × 1.0 = 1.0%

Потеря: 1.0% × 510 = 5.1 прочности
Итог: 505/510 (Pristine → Pristine)

═══════════════════════════════════════════════════════════════════

При 10 ударах L9 техникой:
  Суммарный урон: 180,000
  Потеря: 10% × 0.556 = 5.56%
  
  Итог: 482/510 (Pristine → Excellent)

═══════════════════════════════════════════════════════════════════

При 100 ударах L9 техникой:
  Суммарный урон: 1,800,000
  Потеря: 100% × 0.556 = 55.6%
  
  Итог: 226/510 (Excellent → Damaged)
  Требуется ремонт!
```
