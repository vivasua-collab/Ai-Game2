## 🎯 ГЛАВНАЯ ЗАДАЧА: Система Тела

### Обзор

Система тела находится в **переходном состоянии** — базовая реализация существует, но не доведена до логической завершённости.

- Новая механика Qi Buffer: сырая Ци поглощает только **90%** урона
- Броня становится **обязательной** для всех уровней
- Щитовая техника получает критическую важность (100% vs 90%)

| Метрика | Старое (100%) | Новое (90%) |
|---------|---------------|-------------|
| Урона за удар | 0 | 3,932 |
| Паралич торса | ∞ | 1-2 удара |
| Нужна ли броня? | Опционально | **ОБЯЗАТЕЛЬНО** |

### Иерархия защиты

```
1. Уклонение (50% кап) → не получить удар
2. Щит-техника (100%) → 0 урона в HP
3. Сырая Ци (90%) → снижение урона
4. Броня (DR%) → дополнительное снижение
5. HP тела → последний рубеж
```
