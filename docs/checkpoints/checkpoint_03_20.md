**Выбранное решение:** Базовое Ци = Ци практики × qiDensity

**Детали:** `docs/checkpoints/checkpoint_03_20_technique.md`

**Ключевые изменения:**
```typescript
// combat.ts - handleTechniqueUse
const baseQiInput = qiCost * qiDensity;
const stability = checkDestabilization(baseQiInput, capacity);
const damage = stability.effectiveQi * statMult * masteryMult * rarityMult;
```
