## 📋 Обзор

Все объекты мира разделены на две категории:

| Категория | Название | Контроллер | Примеры |
|-----------|----------|------------|---------|
| **Живые** | SoulEntity | `player` / `ai` | Игрок, NPC, монстры, духи, големы |
| **Неживые** | PhysicalObject | `none` | Камни, руды, деревья, строения |

```
┌─────────────────────────────────────────────────────────────────────────┐
│                     ВСЕ ОБЪЕКТЫ МИРА                                     │
├─────────────────────────────────────────────────────────────────────────┤
│   ┌─────────────────────────────────┐   ┌─────────────────────────────┐ │
│   │     ЖИВЫЕ (SoulEntity)          │   │   НЕЖИВЫЕ (PhysicalObject)  │ │
│   ├─────────────────────────────────┤   ├─────────────────────────────┤ │
│   │   ✅ Игрок, NPC, монстры        │   │   ⬜ Камни, руды            │ │
│   │   ✅ Духи, големы               │   │   ⬜ Деревья, растения      │ │
│   │   ✅ Духовные артефакты         │   │   ⬜ Строения, декорации    │ │
│   │                                 │   │                             │ │
│   │   Признаки:                     │   │   Признаки:                 │ │
│   │   • Душа (Soul)                 │   │   • Нет души                │ │
│   │   • Контроллер (player/ai)      │   │   • controller='none'       │ │
│   │   • Body + Qi + Mind            │   │   • Только физика           │ │
│   └─────────────────────────────────┘   └─────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────┘
```

## 1️⃣ SoulEntity (Живые объекты)

### Архитектура

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          SOUL ENTITY (ЖИВАЯ СУЩНОСТЬ)                   │
├─────────────────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │                       SOUL CORE (ОБЯЗАТЕЛЬНО)                    │   │
│  │  id | soulType | controller (player/ai) | status                │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                              │                                           │
│              ┌───────────────┼───────────────┐                          │
│              ▼               ▼               ▼                          │
│  ┌───────────────┐  ┌───────────────┐  ┌───────────────┐               │
│  │     BODY      │  │      QI       │  │    MIND       │               │
│  │  (опционально)│  │  (обязательно)│  │  (опционально)│               │
│  └───────────────┘  └───────────────┘  └───────────────┘               │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  ATTACHMENTS: techniques[] | inventory[] | effects[]             │   │
│  └─────────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────────┘
```

### Типы живых объектов

| Тип | Body | Qi | Mind | Примеры |
|-----|------|-----|------|---------|
| `character` | ✅ organic | ✅ core | ✅ full | Игрок, NPC |
| `creature` | ✅ organic | ✅ core | ⚠️ instinct | Монстры |
| `spirit` | ❌/ethereal | ✅ reservoir | ✅ full | Духи, призраки |
| `artifact` | ✅ mineral | ✅ reservoir | ⚠️ simple | Разумный меч |
| `construct` | ✅ construct | ✅ reservoir | ⚠️ simple | Голем |

### Реализация

**Файлы:**
- `src/types/soul-entity.ts` — TypeScript интерфейсы
- `src/lib/game/soul-factory.ts` — Фабрика создания
- `src/lib/game/controller-service.ts` — Управление контроллером

**База данных:**
- `prisma/schema.prisma` → модель `SoulEntity`

## 2️⃣ PhysicalObject (Неживые объекты)

### Архитектура

```
┌─────────────────────────────────────────────────────────────────────────┐
│                     PHYSICAL OBJECT (НЕЖИВОЙ ОБЪЕКТ)                    │
├─────────────────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │                     BASE (ОБЯЗАТЕЛЬНО)                           │   │
│  │  id | name | category (resource/structure/decoration/item)       │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                              │                                           │
│              ┌───────────────┼───────────────┐                          │
│              ▼               ▼               ▼                          │
│  ┌───────────────┐  ┌───────────────┐  ┌───────────────┐               │
│  │    PHYSICS    │  │   DURABILITY  │  │   QI STORE    │               │
│  │  (обязательно)│  │   (опц.)      │  │   (опц.)      │               │
│  └───────────────┘  └───────────────┘  └───────────────┘               │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  CONTENTS: resources[] | inventory[] | interactions[]            │   │
│  └─────────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────────┘
```

### Категории

| Категория | Взаимодействие | Примеры |
|-----------|----------------|---------|
| `resource` | mine, harvest, chop | Руда, дерево, трава |
| `structure` | - | Стены, здания |
| `container` | open | Сундуки |
| `decoration` | - | Камни, статуи |
| `item` | use | Предметы |

### Реализация

**Файлы:**
- `src/types/physical-object.ts` — TypeScript интерфейсы
- `src/lib/game/physical-object-factory.ts` — Фабрика создания

**База данных:**
- `prisma/schema.prisma` → модель `PhysicalObject`

## 3️⃣ Сравнение систем

| Критерий | SoulEntity | PhysicalObject |
|----------|------------|----------------|
| **Душа** | ✅ Есть | ❌ Нет |
| **Контроллер** | player / ai | none |
| **Действия** | Осознанные | Реактивные |
| **Движение** | Самостоятельное | Неподвижны |
| **Qi** | core или reservoir | qiStore (опц.) |
| **Техники** | ✅ Используют | ❌ Не могут |
| **Инвентарь** | ✅ Носит | ✅ Может содержать |
