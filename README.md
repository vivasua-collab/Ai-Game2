# 🌸 Cultivation World Simulator

Симулятор мира культивации — современное веб-приложение для управления персонажем в мире, вдохновлённом восточными фэнтези-играми с системой Ци.

## 📋 Описание проекта

Cultivation World Simulator — это интерактивная игра-симулятор, где игроки управляют персонажем-культиватором. Основные возможности:

- **Система культивации** — медитация, накопление Ци, прорывы на новые уровни
- **Управление персонажем** — здоровье, энергия, аспекты культивации
- **Система техник** — активные способности с уровнями и влиянием характеристик
- **Навыки культивации** — пассивные способности для улучшения практики
- **Формации** — защитные конструкции для медитации
- **Исследование мира** — локации с 3D координатами, события, взаимодействия
- **Динамическое время** — внутриигровое время с сменой времени суток
- **Система усталости** — реалистичное управление ресурсами персонажа

## 🏗️ Архитектура

### Ключевые принципы

1. **Сервер — источник истины** — все расчёты на сервере
2. **Разделение ответственности** — services, lib/game, stores
3. **Единый источник констант** — все константы в constants.ts

### Структура проекта

```
src/
├── app/api/          # API эндпоинты
├── components/       # React компоненты
├── data/presets/     # Пресеты данных
├── hooks/            # React хуки
├── lib/game/         # Игровая логика
├── services/         # Сервисный слой
├── stores/           # Zustand хранилища
└── types/            # TypeScript типы
```

## 🆕 Новые системы (v4)

### Уровни техник
- Каждая техника имеет уровень (1-9)
- Штраф за изучение техник выше уровня персонажа
- Влияние характеристик на эффективность

### Навыки культивации
- Пассивные способности (deep_meditation, qi_perception, etc.)
- Сохраняются в БД как JSON
- Улучшают медитацию и поглощение Ци

### Система обучения
- Пресеты (автоматически)
- NPC (обучение у наставника)
- Свитки (изучение предметов)
- Прозрение (создание новых техник)

### 3D координаты
- x, y, z для всех объектов мира
- Высота влияет на доступность
- 2D карта для игрока

### Усталость от уровня
- 9-й уровень культивации — почти не устаёт
- Множители восстановления до x100
- Множители накопления до x0.01
