## 📦 Контейнеры и окна

### Диалог (Dialog / Modal / Модальное окно)
```
┌─────────────────────────────────────────────────────┐
│  📋 Заголовок                              [X]      │  ← Header
├─────────────────────────────────────────────────────┤
│                                                     │
│                  Содержимое                         │  ← Body/Content
│                                                     │
└─────────────────────────────────────────────────────┘
```
- **Dialog** — компонент из shadcn/ui
- **Модальное окно** — общее название
- **DialogContent** — контейнер содержимого
- **DialogHeader** — заголовок окна
- **DialogTitle** — текст заголовка

### Панель (Panel)
```
┌──────────────────┐
│  Заголовок       │
├──────────────────┤
│                  │
│    Содержимое    │
│                  │
└──────────────────┘
```
- Часть интерфейса, занимающая область экрана
- Может быть внутри диалога или самостоятельной

### Вкладки (Tabs)
```
┌────────┬────────┬────────┐
│ Вкладка│ Вкладка│ Вкладка│  ← TabsList
│   1    │   2    │   3    │
├────────┴────────┴────────┤
│                           │
│   Содержимое вкладки      │  ← TabsContent
│                           │
└───────────────────────────┘
```
- **TabsList** — контейнер кнопок вкладок
- **TabsTrigger** — кнопка вкладки
- **TabsContent** — содержимое вкладки

## 🎮 Элементы управления

### Кнопка (Button)
```
┌──────────────┐
│   Нажать     │
└──────────────┘
```
- Активный элемент для действия
- Варианты: primary, secondary, outline, ghost

### Поле ввода (Input)
```
┌──────────────────────┐
│ Введите текст...     │
└──────────────────────┘
```

### Выпадающий список (Select / Dropdown)
```
┌────────────────┐  ▼
│   Выбрано      │
└────────────────┘
```

### Переключатель (Switch / Toggle)
```
○──────   или   ●──────
```

### Флажок (Checkbox)
```
☐ Не выбрано    ☑ Выбрано
```

## 📊 Отображение данных

### Карточка (Card)
```
┌─────────────────────────┐
│  Заголовок              │
│  ─────────────────────  │
│  Описание или данные    │
│  ─────────────────────  │
│  Действия               │
└─────────────────────────┘
```

### Бейдж (Badge)
```
┌──────────┐
│  Текст   │
└──────────┘
```
- Маленькая метка для статусов, категорий

### Прогресс-бар (Progress Bar)
```
████████████░░░░░░░░  60%
```
- Визуализация прогресса

### HP полоска (HP Bar)
```
████████████░░░░░░░░  120/200 HP
```
- Специальный прогресс-бар для очков здоровья

### Ячейка (Slot)
```
┌──────┐
│  🗡️  │
│  x5  │
└──────┘
```
- Квадратная ячейка для предмета инвентаря

### Сетка (Grid)
```
┌────┬────┬────┬────┐
│    │    │    │    │
├────┼────┼────┼────┤
│    │    │    │    │
├────┼────┼────┼────┤
│    │    │    │    │
└────┴────┴────┴────┘
```
- Множество ячеек, организованных в строки и столбцы

## 🖼️ Специфичные элементы проекта

### Кукла тела (Body Doll)
```
      ○ ← Голова
     ╱|╲
    ╱ │ ╲ ← Руки
   ●  │  ●
  │   │   │
  │   ❤️  │ ← Сердце (внутри торса)
  │   │   │
  ║───┼───║ ← Торс
  │   │   │
  ║   │   ║
  ║   │   ║ ← Ноги
  │   │   │
 ○   │   ○
└─── ┴ ───┘
  ↑     ↑
Стопы
```
- SVG-схематичное изображение тела
- Кликабельные части
- Цветовая индикация состояния

### Часть тела (Body Part)
- Голова, торс, рука, нога и т.д.
- Имеет HP (функциональную и структурную)
- Имеет статус (healthy, damaged, crippled, paralyzed, critical, severed)

### Тултип (Tooltip)
```
        ┌─────────────────┐
        │  Информация     │
        │  о элементе     │
        └────────┬────────┘
                 ▼
            [Элемент]
```
- Всплывающая подсказка при наведении

### Скролл-область (Scroll Area)
```
┌─────────────────┐
│   Элемент 1     │
│   Элемент 2     │
│   Элемент 3     │  ▲
│   Элемент 4     │  █ ← Скроллбар
│   Элемент 5     │  ▼
└─────────────────┘
```
- Область с прокруткой

## 🎨 Цветовая индикация статусов

### Состояния частей тела
| Статус | Цвет | Описание |
|--------|------|----------|
| healthy | 🟢 Зелёный | Здорова |
| damaged | 🟡 Жёлтый | Повреждена |
| crippled | 🟠 Оранжевый | Изуродована |
| paralyzed | 🔴 Красный | Парализована |
| critical | 🔴 Красный (пульс) | Критическое |
| severed | ⚫ Серый | Отрублена |

### Редкость предметов
| Редкость | Цвет | CSS класс |
|----------|------|-----------|
| common | Серый | border-slate-500 |
| uncommon | Зелёный | border-green-500 |
| rare | Синий | border-blue-500 |
| epic | Фиолетовый | border-purple-500 |
| legendary | Золотой | border-amber-500 |

## 📝 Примеры использования терминологии

### Правильно ✅
- "Расширь **модальное окно** инвентаря"
- "Увеличь **панель** куклы тела"
- "Добавь **тултип** к ячейкам инвентаря"
- "Измени цвет **бейджа** статуса"

### Неправильно ❌
- "Сделай окно больше" (какое окно?)
- "Растяни панельку" (какую?)
- "Добавь подсказку" (тултип или текст?)
