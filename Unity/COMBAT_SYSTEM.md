## 💥 Расчёт урона

### Базовый урон техники

**Формула:**
```
finalDamage = capacity × gradeMult
```

Где:
- `capacity` = затраченное Ци (ограничено ёмкостью техники)
- `gradeMult` = множитель Grade (×1.0 ~ ×1.6)

### Модификаторы урона

| Модификатор | Формула |
|-------------|---------|
| Мастерство | +0.5% за каждый % мастерства |
| Проводимость | +0.3% за каждую единицу проводимости |
| Уровень культивации | × множитель уровня |
| Стихийный бонус | Зависит от стихии |

## 📊 Подавление уровнем (Level Suppression)

### Проблема

Qi Buffer с поглощением 90% позволяет практике L1 наносить урон практике L8.

### Решение

Множитель подавления на основе разницы уровней:

| Разница | Normal | Technique | Ultimate |
|---------|--------|-----------|----------|
| 0 | ×1.0 | ×1.0 | ×1.0 |
| 1 | ×0.5 | ×0.75 | ×1.0 |
| 2 | ×0.1 | ×0.25 | ×0.5 |
| 3 | ×0.0 | ×0.05 | ×0.25 |
| 4 | ×0.0 | ×0.0 | ×0.1 |
| 5+ | ×0.0 | ×0.0 | ×0.0 |

### Типы атак

| Тип | Описание |
|-----|----------|
| **normal** | Обычная атака без техники |
| **technique** | Атака техникой (technique.level влияет) |
| **ultimate** | Ultimate-техника (особый флаг isUltimate) |

### Формула

```
effectiveAttackerLevel = max(attackerLevel, techniqueLevel)  // для техник
levelDiff = max(0, defenderLevel - effectiveAttackerLevel)
suppression = SUPPRESSION_TABLE[min(5, levelDiff)][attackType]
finalDamage = rawDamage × suppression
```

## 🩸 Применение урона к телу

### Поток повреждения

1. **Функциональная HP** снижается первой
2. При 0 функциональной → урон идёт в структурную
3. При 0 структурной → часть отрублена

### Кровотечение

При значительном уроне начинается кровотечение:

| Степень | HP/тик |
|---------|--------|
| Лёгкое | 1 |
| Среднее | 3 |
| Тяжёлое | 5 |
| Критическое | 10 |

## 🎮 Механика боя

### Инициатива

Бой происходит в реальном времени с возможностью паузы.

### Действия в бою

| Действие | Стоимость времени |
|----------|-------------------|
