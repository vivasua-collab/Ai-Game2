# 🌸 Cultivation World Simulator

Симулятор мира культивации — современное веб-приложение для управления персонажем в мире, вдохновлённом восточными фэнтези-играми с системой Ци.

## 📋 Описание проекта

Cultivation World Simulator — это интерактивная игра-симулятор, где игроки управляют персонажем-культиватором. Основные возможности:

- **2D мир на Phaser 3** — перемещение персонажа, исследование локаций, тренировочный полигон
- **Система культивации** — медитация, накопление Ци, прорывы на новые уровни
- **Система техник** — активные способности с уровнями и влиянием характеристик
- **Навыки культивации** — пассивные способности для улучшения практики
- **Формации** — защитные конструкции для медитации
- **Исследование мира** — локации с координатами, события, взаимодействия
- **Динамическое время** — внутриигровое время с сменой времени суток
- **Система усталости** — реалистичное управление ресурсами персонажа

## 🏗️ Архитектура

### Ключевые принципы

1. **Память первична** — активная сессия в TruthSystem, БД для persistence
2. **Сервер — источник истины** — все расчёты на сервере
3. **Разделение ответственности** — services, lib/game, stores
4. **Единый источник констант** — все константы в constants.ts

### Структура проекта

```
src/
├── app/api/          # API эндпоинты
├── components/
│   ├── game/         # PhaserGame, ChatPanel, MeditationDialog
│   └── ui/           # shadcn/ui компоненты
├── data/presets/     # Пресеты данных
├── hooks/            # React хуки
├── lib/game/         # Игровая логика
├── services/         # Сервисный слой
├── stores/           # Zustand хранилища
└── types/            # TypeScript типы
```

📖 **Подробная архитектура:** [docs/ARCHITECTURE.md](./docs/ARCHITECTURE.md)

## 📡 API

### Основные эндпоинты

| Эндпоинт | Метод | Описание |
|----------|-------|----------|
| `/api/game/start` | POST | Начать новую игру |
| `/api/game/state` | GET | Получить состояние игры |
| `/api/game/move` | POST | Движение + время + пассивное Ци |
| `/api/chat` | POST | Отправить действие/сообщение |
| `/api/rest` | POST | Медитация, отдых, сон |
| `/api/technique/use` | POST | Использование техники |
| `/api/technique/slot` | POST | Назначение техники в слот |
